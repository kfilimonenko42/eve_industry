# eve industry

This project can help you get information about materials, prices, and production stages (eve online)

## Building

### Windows

```cmake
mkdir build
cd build
cmake .. -G "Visual Studio 17 2022"
cmake --build . --config Release
```

### Linux

```cmake
mkdir release
cd release
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build . 
```
